# Dự án 5: Ứng dụng Ghi chú (Provider State)

## Mô tả
Ứng dụng ghi chú với quản lý state toàn cục sử dụng Provider pattern.

## Tính năng
- ✅ Tạo ghi chú mới
- ✅ Chỉnh sửa ghi chú
- ✅ Xóa ghi chú
- ✅ Chọn màu cho ghi chú
- ✅ Hiển thị dạng lưới (GridView)
- ✅ Provider state management
- ✅ ChangeNotifier pattern
- ✅ Cập nhật real-time

## Cách chạy
```bash
flutter pub get
flutter run
```

## Kỹ thuật sử dụng
- Provider package
- ChangeNotifier
- Consumer widget
- context.read<T>()
- GridView.builder
- TextField multiline
- Color picker custom
